# Gradient Card Reference

**Tanggal disimpan:** 29 Mei 2026  
**Sumber:** Framer / design tool (component preview)  
**File gambar:** `gradient-card-reference.jpg`

## Deskripsi Komponen

Card UI dengan estetika dark-mode premium, cocok untuk feature highlight di landing page SaaS.

## Visual Breakdown

**Struktur card:**
- Background atas: dark navy/charcoal, hampir hitam dengan tint biru gelap
- Background bawah: gradient kaya — teal gelap (kiri) → ungu/violet (tengah-kanan) → pinkish-mauve (kanan bawah)
- Corner radius: besar, rounded rectangle
- Tidak ada hard border atau shadow — efek glow dari gradient itu sendiri

**Konten (atas ke bawah):**
- Icon badge: lingkaran abu gelap dengan ikon bintang putih (kiri atas)
- Heading: bold, putih, besar — contoh: *"AI-Powered Inbox Sorting"*
- Body text: light-weight, putih/abu terang, 2–3 baris
- CTA link: *"Learn More →"* putih, tanpa tombol eksplisit

## Kesan & Penggunaan

Cocok untuk:
- Feature section di landing page
- Highlight fitur premium/paywall
- Komponen "what you get" di halaman pricing

Tone: modern, premium, dark SaaS — konsisten dengan branding Devil's Advocate.
